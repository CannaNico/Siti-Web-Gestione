# Sommario Esecutivo

## Descrizione sintetica del progetto
Realizzazione del sito web ufficiale dell’associazione **Libero Stile**, dedicata ad attività sportive, amicizia, socializzazione ed esperienze condivise.

## Scopo principale dello studio
Valutare la fattibilità tecnica, economica e organizzativa dello sviluppo del sito web associativo.

## Raccomandazione finale
**[FATTIBILE]** — Il progetto richiede investimenti contenuti e offre ottime possibilità di crescita della community e visibilità dell’associazione.

## Investimento stimato
Tra **€1.500** e **€4.000**

## ROI atteso
20% - 35% entro 12-24 mesi

## Rischi principali
- Bassa partecipazione iniziale
- Aggiornamento contenuti non costante
- Costi di gestione eventi
- Scarsa promozione online
- Dipendenza dall’attività dei volontari
