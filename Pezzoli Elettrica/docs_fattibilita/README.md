# Sommario Esecutivo

## Descrizione sintetica del progetto
Realizzazione di un sito web professionale per l’azienda **Pezzoli Elettrica**, specializzata in impianti elettrici civili e industriali. Il sito avrà funzione informativa, promozionale e di contatto con i clienti.

## Scopo principale dello studio
Valutare la fattibilità tecnica, economica e organizzativa dello sviluppo del sito web aziendale.

## Raccomandazione finale
**[FATTIBILE]** — Il progetto presenta costi contenuti, elevata utilità commerciale e buone prospettive di acquisizione clienti.

## Investimento stimato
Tra **€2.000** e **€5.000**

## ROI atteso
25% - 40% **entro 12-18 mesi**

## Rischi principali
- Bassa visibilità iniziale online
- Ritardi nello sviluppo
- Costi di manutenzione futuri
- Aggiornamento contenuti non costante
- Concorrenza locale elevata
